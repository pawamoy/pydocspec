pydocspec.converter

module documentation
(source)

Convert docspec.Module objects into a pydocspec.ApiObjectsRoot containing all your objects.

This converter is supposed to be fully compatible with docspec_python.

Usage:

    import pydocspec
    from docspec_python import load_python_modules
    from pydocspec.converter import convert_docspec_modules
    root: pydocspec.ApiObjectsRoot = convert_docspec_modules(load_python_modules(...))
NoteIt will transform the tree such that we have an actual hiearchy of packages.
